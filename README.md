# My Personal Website

A simple personal website made for **CMSC 207 – Web Programming and Development**. Inspired by *Create with Code: Build Your Own Website* by Clyde Hatter. It includes information about me, my hobbies, and how to contact me.

## Project Objectives

- Develop a structured and semantic HTML page
- Apply CSS to enhance layout, colors, and typography
- Use JavaScript for simple interactivity
- Implement responsive design for mobile and desktop views
- Get comfortable with debugging and refining web pages

## Website Sections

| Section | Description |
|---|---|
| **Header** | Name, site title, and navigation menu |
| **About Me** | Short introduction with an image |
| **Hobbies & Interests** | At least three hobbies with brief descriptions |
| **Contact Info** | Email address and social media links |

## Built With

- HTML — semantic tags (`<header>`, `<section>`, `<footer>`, etc.)
- CSS — external stylesheet for design, layout, and media queries
- JavaScript — simple interactivity (e.g., fun fact button, hover effects)

## Features

- Navigation menu with working anchor links
- Hover effects on buttons and images
- JavaScript button that displays a fun fact
- Mobile-friendly layout using CSS media queries

## How to View

1. Download or clone this repository
2. Open `index.html` in your browser

> All assets (images, etc.) are in the same folder as `index.html` — no additional setup needed.
